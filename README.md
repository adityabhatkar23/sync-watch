# SyncWatch

Minimal real-time video sync app.

Watch locally downloaded videos together over the internet.  
One user acts as host and controls play, pause, and seek.

## Stack

- React (Vite)
- Node.js + Express
- Socket.io

Only playback events are synced. Video files are not shared.
